# Lost Base Shop Bot

بوت متجر لوست بيس لديسكورد — نظام متكامل للشراء بـ لوست كوين

---

## الملفات المطلوبة في الهوست

أنشئ ملف `.env` في المجلد الرئيسي وضع فيه:

```env
# توكن البوت من Discord Developer Portal
DISCORD_TOKEN=your_discord_bot_token_here

# ID البوت (Client ID)
CLIENT_ID=your_bot_client_id_here

# ID السيرفر (للنشر السريع للأوامر)
GUILD_ID=your_guild_id_here

# قاعدة البيانات MongoDB
MONGODB_URI=mongodb+srv://2026:123@cluster0.fmnowx6.mongodb.net/?appName=Cluster0

# ID الكاتاقوري اللي تفتح فيها التكتات
SHOP_CATEGORY_ID=1492286988261523678

# ID رتبة الإدمن اللي تقدر تضيف منتجات
ADMIN_ROLE_ID=1494667108833951957

# ID قناة اللوق
SHOP_LOG_CHANNEL_ID=1494668694528327750

# ID حساب المحفظة (اللي يتحول له الكردت)
PROBOT_ACCOUNT_ID=598583098330054664
```

## طريقة التشغيل

```bash
# تثبيت المتطلبات
npm install

# نشر الأوامر على السيرفر
node src/deploy-commands.js

# تشغيل البوت
npm start
```

## الأوامر المتاحة

| الأمر | الوصف | من يستخدمه |
|-------|--------|------------|
| `/shoppanel` | إرسال لوحة المتجر الثابتة | إدمن |
| `/add-item [name] [price] [quantity]` | إضافة منتج جديد | رتبة الإدمن |
| `/add-quantity [item] [quantity]` | زيادة كمية منتج | رتبة الإدمن |
| `/give-item [user] [item]` | إهداء منتج لعضو | رتبة الإدمن |
| `/buy-coins` | عرض طريقة شراء لوست كوين | الجميع |
| `/balance` | عرض رصيد الكوين | الجميع |
| `/balance [user]` | عرض رصيد عضو معين | رتبة الإدمن |

## نظام لوست كوين

**سعر الصرف:** 10,000 كردت = 1 لوست كوين

**طريقة الشراء للأعضاء:**
1. يستخدم الأمر `/buy-coins` ليشوف التعليمات
2. يحول الكردت لـ ProBot:
   ```
   c 598583098330054664 [المبلغ]
   ```
3. البوت يكتشف رسالة التحويل تلقائيًا ويضيف الكوينز

---

## كيف يعمل نظام التحويل التلقائي

- البوت يراقب رسائل ProBot (ID الرسمي: `282859044593598464`)
- لما يشوف رسالة تحويل للحساب `598583098330054664`، يحسب الكوينز ويضيفها
- يرسل فاتورة على DM للشخص
- يسجل العملية في قناة اللوق
